# Template Notes

## Assumptions

- 기본 담당자 예시는 your-name이다.
- 기본 날짜 예시는 2026-05-12이다.
- 문서 기본 언어는 Korean이다.
- 문서 파일은 Markdown, 메타데이터는 YAML을 사용한다.
- 파일명은 lower-kebab-case를 기준으로 한다.
- 이 저장소는 실제 제품 코드가 아니라 프로젝트 초기화를 위한 템플릿 팩이다.

## Usage Notes

템플릿을 복사한 뒤 프로젝트별 실제 값으로 변수를 치환해야 한다. 아직 확정되지 않은 값은 단정하지 말고 assumptions.md 또는 Agent 상태 문서에 근거와 함께 기록한다.

# Roadmap

## 현재 안정화 범위

- 템플릿 생성기 안정화
- 모든 템플릿 smoke test
- Delivery Sanitization
- 기존 프로젝트 적용 스크립트
- 공개 저장소 기본 문서 보강

## 다음 후보 작업

- 템플릿별 더 정교한 delivery policy preset
- 기존 프로젝트 적용 시 충돌 병합 전략 확장
- 생성 프로젝트용 CI scaffold 옵션
- 보안/규제 템플릿의 감사 증적 보존 정책 강화
- production-agent-system의 실제 provider adapter 예시 추가

## 보류 항목

- Git history sanitization 자동화
- 감사 증적 자동 삭제
- 라이선스 파일 자동 제외

위 항목은 계약과 법무 검토가 필요한 영역이므로 자동화하지 않는다.

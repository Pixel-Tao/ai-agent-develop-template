# Greenfield Basic

Korean is the default documentation language. This file is the English companion for [README.md](README.md).

## Purpose

A basic AI-agent friendly structure for new development projects. Copy this template into a real project, replace variables, and fill the documents for the specific context.

## Use When

- Starting a new project
- Working with a small team of one to three people
- You need fast initial setup

## Do Not Use When

- The project already has a complex codebase
- Large-scale role separation is required

## Initialization

1. Fill project metadata in `manifest.yaml`.
2. Read `AGENTS.md`.
3. Record the current state in `docs/09_agent_state/current-status.md`.
4. Select required skills from `skills/skills-index.yaml`.
5. Add initial work to backlog or task documents.

## Required Entrypoints

- `README.md`
- `AGENTS.md`
- `manifest.yaml`
- `template.yaml`
- `validation-checklist.md`
- `docs/09_agent_state/current-status.md`

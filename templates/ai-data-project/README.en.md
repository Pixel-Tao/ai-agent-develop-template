# AI Data Project

Korean is the default documentation language. This file is the English companion for [README.md](README.md).

## Purpose

A structure for AI, data, LLM, model evaluation, prompt, and experiment projects.

## Use When

- The project includes LLM/RAG work
- Model or prompt evaluation matters
- Data pipelines and experiment tracking are important

## Do Not Use When

- The project is a general CRUD app
- Data and evaluation are not central to the work

## Initialization

1. Fill project metadata in `manifest.yaml`.
2. Review data, model, prompt, and evaluation documents.
3. Register datasets, prompts, experiments, and failure cases.
4. Record current state in `docs/09_agent_state/current-status.md`.

## Required Entrypoints

- `README.md`
- `AGENTS.md`
- `manifest.yaml`
- `template.yaml`
- `validation-checklist.md`
- `docs/09_agent_state/current-status.md`

# Task: Initial Setup

템플릿을 실제 프로젝트에 적용한 직후 수행할 초기 설정 작업이다. 문서 변수 치환, 상태 문서 작성, 초기 backlog 등록을 완료 기준으로 둔다.

## 사용 방법

이 문서는 프로젝트 초기화 시 초안을 작성하고 작업 진행에 따라 갱신한다. AI Agent는 작업 전에 이 문서의 현재 상태를 확인하고, 불확실한 내용은 단정하지 않고 assumptions 또는 unknowns에 기록한다.

- [ ] placeholder 치환
- [ ] manifest 갱신
- [ ] Agent 상태 문서 작성
- [ ] 초기 검증 완료

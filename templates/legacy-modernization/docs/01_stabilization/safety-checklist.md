# Safety Checklist

레거시 변경 전 안전장치를 확인한다. 테스트, 백업, 모니터링, rollback, 승인 상태를 함께 점검한다.

## 사용 방법

이 문서는 프로젝트 초기화 시 초안을 작성하고 작업 진행에 따라 갱신한다. AI Agent는 작업 전에 이 문서의 현재 상태를 확인하고, 불확실한 내용은 단정하지 않고 assumptions 또는 unknowns에 기록한다.

- [ ] 핵심 동작 테스트 존재
- [ ] 백업 또는 복구 경로 확인
- [ ] rollback 계획 확인
- [ ] 영향 범위 기록
- [ ] 승인 필요 여부 확인

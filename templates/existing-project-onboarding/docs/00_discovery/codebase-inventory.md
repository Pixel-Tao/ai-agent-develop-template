# Codebase Inventory

코드베이스의 주요 영역, 경로, 역할, 중요도, 테스트 여부를 기록한다. Agent는 변경 전 이 문서를 확인해 영향 범위를 먼저 추정한다.

## 사용 방법

이 문서는 프로젝트 초기화 시 초안을 작성하고 작업 진행에 따라 갱신한다. AI Agent는 작업 전에 이 문서의 현재 상태를 확인하고, 불확실한 내용은 단정하지 않고 assumptions 또는 unknowns에 기록한다.

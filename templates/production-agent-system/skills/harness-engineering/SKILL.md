# Harness Engineering

Use this skill when adding or verifying commands for the generated Agent system.

## Workflow

1. Read `harness/commands.md`.
2. Confirm whether the command is implemented or still draft.
3. Execute only implemented commands.
4. Record command, result, failure cause, and artifact location in `harness/evidence-log.md`.
5. Update `harness/verification-matrix.md` if the minimum verification changes.

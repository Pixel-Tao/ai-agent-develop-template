# Init Archive

초기 설정 완료 후 INIT.md와 관련 기록을 보관하는 위치다. 루트의 INIT.md가 이곳으로 이동된 뒤에는 archived INIT을 자동으로 다시 실행하지 않는다.

초기화 중 루트 README.md를 프로젝트 맞춤 문서로 재작성하기 전, 기존 README는 original-README-{{YYYY-MM-DD}}.md 형식으로 이 폴더에 보관한다. 같은 날짜 파일이 이미 있으면 suffix를 붙이고 덮어쓰지 않는다. init-archive-log.md에는 INIT 아카이브와 함께 README 보관 위치 및 README.en.md 동기화 상태를 기록한다.

재초기화가 필요하면 먼저 docs/09_agent_state/current-status.md, run-log.md, assumptions.md를 확인하고 사용자에게 재초기화 의도를 확인한다.

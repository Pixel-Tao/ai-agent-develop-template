# Initialization Prompt

Use `INIT.md` as the active one-time initialization prompt. This file documents that the generated project expects an Agent system discovery interview before runtime implementation work starts.

During initialization, archive the original root `README.md` to `docs/09_agent_state/archive/init/original-README-{{YYYY-MM-DD}}.md` before rewriting it. The final `README.md` must describe the generated Agent system project, not the template, and `README.en.md` must be synchronized or marked as needing an update.

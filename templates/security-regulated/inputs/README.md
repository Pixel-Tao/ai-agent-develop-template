# Inputs

## 목적

이 폴더는 프로젝트를 시작하기 전에 사용자가 이미 가지고 있는 초기 개발 문서, 아이디어 메모, 요구사항 초안, 화면 스케치, 회의록, 레퍼런스 링크를 넣어두는 공간이다. AI Agent는 구현이나 인터뷰를 시작하기 전에 이 폴더를 먼저 확인하고, 문서에서 확인된 사실과 추가 질문이 필요한 내용을 분리해야 한다.

## 사용 방법

- initial-development-docs/에는 사용자가 제공한 원본 문서를 넣는다.
- references/에는 레퍼런스 링크, 경쟁 제품 메모, 외부 자료 요약을 넣는다.
- source-documents-index.md에는 각 문서의 목적, 출처, 신뢰도, 반영 상태를 기록한다.
- 원본 문서는 가능한 수정하지 말고, 해석과 요약은 별도 문서에 남긴다.

## Agent 규칙

Agent는 이 폴더를 읽고 바로 요구사항으로 확정하지 않는다. 문서에서 확인된 사실, 사용자의 의도처럼 보이는 내용, 추정, 충돌, 미정 질문을 구분한 뒤 Project Discovery Interview에서 사용자에게 확인한다.

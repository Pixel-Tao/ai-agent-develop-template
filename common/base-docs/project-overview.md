# Project Overview

프로젝트의 목적, 대상 사용자, 핵심 기능, 현재 상태를 한 곳에 정리하는 문서다. 신규 참여자와 AI Agent는 이 문서를 통해 프로젝트가 왜 존재하는지와 어디까지가 범위인지 먼저 이해한다.

## 사용 방법

이 문서는 프로젝트 초기화 시 초안을 작성하고 작업 진행에 따라 갱신한다. AI Agent는 작업 전에 이 문서의 현재 상태를 확인하고, 불확실한 내용은 단정하지 않고 assumptions 또는 unknowns에 기록한다.
